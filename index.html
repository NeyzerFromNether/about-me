NeuronOS

Автономная SaaS платформа для управления Telegram каналами с AI-генерацией контента.

Нейросеть, которая думает. Анализирует. Постит. Учится. Работает полностью автономно — без кнопок "Сгенерировать" и "Опубликовать". AI анализирует метрики каналов и адаптирует контент под аудиторию.

Возможности:

  • 🧠 Живой AI — автономная генерация контента
  • 📊 Самообучение — AI анализирует метрики и адаптируется
  • 💀 Вирусный контент — генерация с анти-AI фильтром
  • ⏰ Планировщик — гибкая настройка интервалов постинга
  • 📈 Аналитика — отслеживание эффективности в реальном времени
  • 🎨 Dark UI — премиум интерфейс в тёмной теме

Категории контента:

Dark Psychology, Relationships, Manipulation, Money Psychology, Provocative, Brain Facts

Архитектура:

Backend: FastAPI + SQLAlchemy + SQLite | Frontend: HTML/CSS/JS с Dark Theme | AI: g4f (бесплатный)

Python FastAPI SQLite g4f Telegram Bot API HTML/CSS/JS

Telegram News Aggregation Bot

Бот для сбора новостей о происшествиях с Telegram/VK каналов с AI-обработкой.

Возможности:

  • 📡 Мониторинг Telegram каналов — сбор постов по ключевым словам (происшествия, ЧП, жесть)
  • 🤖 Фильтрация контента — AI (Ollama) проверяет релевантность постов
  • ✍️ Рерайт постов — переписывает текст в стиле популярных каналов
  • ✅ Модерация — пользователи оценивают посты через кнопки 👍/👎/📝
  • 💬 Обратная связь — накопление причин отклонений для улучшения фильтрации
  • 🔄 Дубликация — проверка на уникальность через хеширование

Структура БД:

newsbot.db | Таблица generated_posts — сгенерированные посты | Таблица feedback — оценки пользователей

Команды бота:

/start — начать работу | /scan — запустить сканирование | /stop — остановить | /admin — панель админа

AudioGeek Bot

Telegram-бот для поиска, скачивания и управления музыкой.

Возможности:

  • 🔍 Поиск музыки — поиск треков по названию или исполнителю
  • ⬇️ Скачивание — скачивание аудио в чат
  • 📋 Плейлисты — создание, удаление и шаринг плейлистов
  • ❤️ Избранное — лайк треков для персональной подборки
  • 🌊 Твоя Волна — персонализированные рекомендации
  • 🎤 Shazam — распознавание треков по голосовому сообщению
  • 📊 Топ Чарт — ежедневные подборки по жанрам
  • 🌍 Мультиязычность — RU, EN, KK, KY, UK, FR, ES, ID, HI, TR (10 языков)

Админ-панель:

  • 📈 Статистика — количество пользователей, пол, страна, рост
  • 📣 Рассылка — отправка сообщений по полу/языку
  • 🎯 Кампании — управление реферальными ссылками
  • 🔗 Шейринг — статистика приглашений
